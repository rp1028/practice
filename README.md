| 워드클라우드 시각화 | 뉴스 키워드 빈도 기반 워드클라우드 이미지 제공 | |

## 2. 비기능 요구사항

| 구분 | 요구사항명 | 상세 설명 | 비고 |
| :---: | :--- | :--- | :--- |
| 성능 | 데이터 캐싱 | 반복 조회 시 재처리 없이 빠른 표시 | |
| 가용성 | DB 없이 동작 | MySQL 미연결 시 CSV 기반으로 정상 동작 | |
| 보안 | API 키 보호 | API 키·DB 접속 정보 외부 미노출 관리 | .env, .gitignore |
| 유지보수성 | 데이터·UI 분리 | 데이터 처리와 화면 렌더링 모듈 분리 | |
| 데이터 품질 | 지역명 표준화 | 모든 소스의 지역명을 17개 표준 시도명으로 통일 | |
| UI/UX | 사이드바 네비게이션 | 사이드바를 통한 7개 메뉴 이동 | |
| UI/UX | 필터 및 검색 | 지역·기간·차종 등 조건 필터 제공 | |
